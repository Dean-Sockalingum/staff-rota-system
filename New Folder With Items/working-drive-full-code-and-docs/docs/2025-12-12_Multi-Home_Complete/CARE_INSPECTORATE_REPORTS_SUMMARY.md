## Integration with Service Improvement Plan System

### Step 1: Import Reports
Upload PDF reports to system, extract key data:
- CS Number (links to home)
- Inspection date
- Ratings for each theme
- Requirements list
- Recommendations list
- Areas for improvement

### Step 2: Auto-Generate Actions
For each requirement/recommendation, create ImprovementAction:
- Title: Requirement/recommendation text
- Source: INSPECTION
- Priority: CRITICAL (requirements) or HIGH (recommendations)
- Quality Theme: Map to CI framework (Care/Environment/Staffing/Management)
- Target completion: Based on CI deadline (typically 3-6 months)

### Step 3: Link to Evidence
As actions progress, link to:
- Training records (staff development)
- Compliance checks (medication audits)
- Care plan reviews (quality metrics)
- Complaint resolutions (feedback improvements)

### Step 4: Track Outcomes
At next inspection:
- Compare ratings: Previous vs Current
- Document improvements: 3 Adequate → 4 Good
- Evidence action completion
- Demonstrate continuous improvement culture
