## ðŸ“‹ CANVA STEP-BY-STEP WORKFLOW

### Phase 1: Setup (10 minutes)

1. Create new Canva Video project (1920x1080)
2. Set duration to 3 minutes
3. Add 11 pages (one per scene)
4. Import color palette (save custom colors)
5. Select fonts (Montserrat, Poppins, Roboto)

### Phase 2: Add Content (45 minutes)

**For Each Scene:**
1. Set page duration (refer to scene durations above)
2. Add background color/image
3. Add text elements (copy from script above)
4. Add icons from Canva library
5. Position elements (use alignment tools)
6. Apply animations:
   - Text: Fade in, Slide up, Typewriter
   - Icons: Bounce, Pop, Zoom
   - Transitions: Dissolve, Slide
7. Preview and adjust timing

### Phase 3: Add Media (30 minutes)

1. Upload screen recordings (via "Uploads" tab)
2. Place recordings on appropriate pages
3. Trim to exact durations needed
4. Add phone/laptop frames (search "mockup frame")
5. Adjust video positioning within frames
6. Add any stock images from Canva library

### Phase 4: Add Audio (30 minutes)

**Background Music:**
1. Click "Audio" tab in Canva
2. Search "corporate background music"
3. Drag onto timeline
4. Set volume to 25%
5. Fade in/out at start/end

**Voiceover:**
- **Option A - Record in Canva:**
  1. Click "Record yourself" button
  2. Read script for each scene
  3. Place audio on correct pages
  
- **Option B - Upload Voice File:**
  1. Record using phone/mic
  2. Upload MP3 to Canva
  3. Place on timeline
  4. Sync with visuals

- **Option C - Use Text-to-Speech:**
  1. Highlight text on page
  2. Click "Convert to audio"
  3. Select voice style
  4. Adjust timing

### Phase 5: Polish (20 minutes)

1. **Preview entire video** (play from start)
2. **Check timing:** Does voiceover match visuals?
3. **Check transitions:** Smooth between scenes?
4. **Check volume:** Music not too loud?
5. **Spelling check:** All text correct?
6. **Brand check:** Colors, fonts consistent?
7. **Mobile preview:** Looks good on all devices?

### Phase 6: Export (5 minutes)

1. Click **"Share"** button (top right)
2. Select **"Download"**
3. Format: **MP4 Video**
4. Quality: **1080p HD**
5. Click **"Download"**
6. Wait for processing (2-5 minutes)
7. Save file as: `Staff_Rota_System_Overview_v1.mp4`

## ðŸ†˜ TROUBLESHOOTING

**Common Canva Issues:**

**Issue:** "Video won't export"  
**Fix:** Reduce resolution to 720p, or split into 2 shorter videos

**Issue:** "Audio out of sync"  
**Fix:** Lock audio to specific page, adjust page duration

**Issue:** "Too much text on screen"  
**Fix:** Break into bullet points, use animations to reveal one at a time

**Issue:** "File size too large"  
**Fix:** Compress video after export using HandBrake (free tool)

**Issue:** "Screen recordings blurry"  
**Fix:** Record at 1080p minimum, ensure browser zoom at 100%
